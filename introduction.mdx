## What You Can Do

### Read Scripture

Access various Bible versions through the API.Bible integration. Navigate seamlessly between chapters with intuitive controls, highlight specific verses with smooth scrolling, and enjoy a responsive design optimized for desktop and mobile devices.

### Search and Discover

Search across Bible text with powerful keyword queries. Navigate through paginated results with search terms highlighted for easy identification. Click on verse references to navigate directly to the text in context.

### Organize Your Study

Create custom collections to organize your favorite verses. Add verses from Bible reading or search results, manage collection names, and remove verses as needed. All your collections are private and tied to your account.

### Take Notes

Create and edit chapter-specific notes using a rich text editor powered by Plate.js. Format your notes with bold, italics, lists, and more. View your notes side-by-side with the Bible text on desktop for an integrated study experience.

### Create AI Stories

Generate unique, creative stories based on biblical passages using Google Gemini AI. Customize story perspective, setting, tone, and length. View original biblical text alongside your generated story, and manage all your stories from a centralized dashboard.

## Technology Stack

Your Bible is built with modern web technologies:

- **TanStack Start** - Full-stack React framework with built-in SSR/SSG
- **React 19** - Latest React with concurrent features
- **TypeScript** - Type-safe development
- **Convex** - Real-time backend with automatic sync
- **Better Auth** - Secure authentication
- **PostgreSQL** - Primary database via Neon
- **Google Gemini AI** - AI-powered story generation
- **Tailwind CSS v4** - Modern, responsive design
- **Shadcn/ui** - High-quality, accessible components
